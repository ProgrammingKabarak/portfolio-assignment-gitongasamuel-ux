# Personal Portfolio

A responsive, minimalist personal portfolio built to showcase my ventures, technical stack, and background in system architecture. Developed as the capstone project for the Markup and Scripting course.

## Author
* **Name:** Samuel Mahianyu Gitonga
* **Institution:** Kabarak University

* acess https://samwebsite13.netlify.app/

## Overview
This portfolio highlights my focus on digital infrastructure and AI orchestration, featuring platforms like Rework, Datarion Systems, and KABI. The interface utilizes a premium, ultra-dark aesthetic and is built entirely from scratch, strictly avoiding external CSS frameworks or JavaScript libraries.

## Tech Stack
* **HTML5:** Semantic structure and accessibility.
* **CSS3:** Custom variables, CSS Grid, and Flexbox for responsive layouts.
* **Vanilla JavaScript:** Interactive components and DOM manipulation.

## Core Features
* **Fully Responsive:** Adapts to mobile, tablet, and desktop environments.
* **Dynamic Filtering:** Interactive category sorting for project ventures.
* **Typing Animation:** Terminal-style text effect in the hero section.
* **Form Validation:** Client-side input validation for the contact section.
* **Scroll Navigation:** Integrated scroll-to-top functionality.
