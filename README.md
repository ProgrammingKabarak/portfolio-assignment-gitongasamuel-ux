Capstone Project: Personal Portfolio Website


Project Overview
github classrom link to accept assignment: https://classroom.github.com/a/QeZzh9gE


Congratulations! You have reached the final assessment for
the Markup and Scripting course. This capstone project is designed to evaluate
the knowledge and practical skills you have acquired throughout the semester.
You are required to design and develop a professional, responsive personal
portfolio website using HTML5, CSS3, and JavaScript.


Your portfolio should showcase your profile, technical
skills, projects, achievements, and contact information while demonstrating
your ability to create well-structured, interactive, and responsive web
applications.


This project is an individual assignment and contributes
significantly to your final course grade.


 


Learning Outcomes


By the end of this project, you should be able to:



• 
 
• Develop
     structured web pages using semantic HTML5.
 
• Apply
     CSS3 to create responsive and visually appealing layouts.
 
• Use
     JavaScript to add interactivity and improve user experience.
 
• Organize
     a web development project professionally.
 
• Apply
     version control using Git and GitHub.
 
• Submit
     projects through GitHub Classroom and Google Classroom.
